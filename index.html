Precision Tracing

SQLite trace storage with metadata-only by default. Total control over what gets logged.

Core Features

Zero Overhead

A transparent proxy designed to run in the background with minimal impact on agent latency.

Smart Redaction

Configurable redaction patterns (OBSERVER_REDACT_PATTERNS) to keep sensitive data out of logs.

Session Isolation

Automatic session isolation and DB permissions (0600) ensure data privacy and integrity.

Raw Payload Opt-in

Use OBSERVER_RAW_PAYLOAD=1 when you need deep inspection of tool inputs and outputs.

Input/Output Hashing

Verify integrity and identify repetitive calls through intelligent payload hashing.

Transparent Proxy

Drop it into your stack and start observing immediately without changing agent logic.

Understand the Flow

From tool calls to trace history, gain full visibility into the Model Context Protocol.
